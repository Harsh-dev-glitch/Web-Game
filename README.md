# Web-Game

A simple browser game built for Hack Clubbers to pass the time, have fun, and challenge their reflexes.

# Controls

Space Bar → Jump

# Objective

Survive as long as possible while avoiding obstacles.

Good luck and try to beat your high score!

# About the Game

This game features:

* Simple physics
* Endless gameplay
* Jump mechanics
* Score tracking
* A giant sun ☀️
* Brown obstacles 🪵

The game is designed to test your timing, rhythm, and reaction speed.

# Built With

* HTML
* CSS
* JavaScript

No frameworks, no game engines — just pure web technologies.
